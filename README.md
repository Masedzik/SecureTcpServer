# SecureTcpServer

SecureTcpServer is a lightweight and secure TCP server designed to handle encrypted communication with multiple client.

## 🚀 Overview

- Accepting multiple client connections
- Receiving and sending encrypted data
- Managing client sessions safely

## 🔐 Features

- Secure TCP server implementation
- Support for encrypted communication
- Multi-client handling
- Event-driven architecture
- Scalable and extensible design

## 🛠️ Technologies

- TCP/IP sockets
- Cryptography
